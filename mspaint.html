Code in MS Paint

Inspired by this post

Takes text input and turns it into decimal color values, correlating with the text's ASCII codes. You can input them in a 1px height (n)px width MS Paint file from left to right, then save as a 24-bit Bitmap. Open it with a text editor like Notepad and you'll see your text in the file. You can enter the colors on a width more than 1px, but be aware that you have to enter them left to right, bottom to top, and that there will be random values inbetween each row of pixels entered. Not all text editors will display line feeds correctly (Notepad didn't for me, but Sublime Text and Vim did).
